Comment l'intégrer dans Gmail ?

1

Cliquez sur « Copier la signature » ci-dessus.

2

Dans Gmail, allez dans Paramètres → Voir tous les paramètres → Général → Signature.

3

Créez une nouvelle signature, puis collez dans la zone de texte (Ctrl+V / Cmd+V).

4

Cliquez sur « Enregistrer les modifications » en bas de page.
